1. A prime number is a prime factor for 1/n of all integers
   2 is a prime factor of a half of all integers
   3 is a prime factor of a third of all integers
2. Numbers which have prime factors are not prime
   we will say they are covered by their prime factor
   2 covers 4, 6, 8, 10, 12, 14, ...
   3 covers 6, 9, 12, 15, 18, 21, ...
3. For a set of primes, the coverages are combined thus:
   1/n0 + (1/n0 * 1/n1) + (1/n0 * 1/n1 * 1/n2) * ...
   The set of [2,3] covers 1/2 + 1/2*1/3 = 2/3 of all numbers
   All primes above 3 are found in the left over 1/3
   The set of [2,3,5] covers 2/3+1/2*1/3*1/5 = 21/30
   All primes above 5 are found in the left over 3/10
4. To the left are prime number sets, empty at first
5. n will count up through all numbers
6. A prime number is added to the bottom set at n/2
   2 is added at 4, 3 is added at 6, ...
7. When a set's coverage exceeds the coverage of the set above it,
   the smallest prime of the lower set is moved to the higher.
8. Each set is assigned a note.
   If that set contains a prime factor of n, that note is played.
